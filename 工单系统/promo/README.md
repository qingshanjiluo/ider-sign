## 📖 平台简介

**艾德尔修仙工单平台**是一个面向《艾德尔修仙传》玩家的在线代练服务平台。  
用户可在网站自助下单，系统通过 GitHub Actions 工作流自动完成账号注册、角色创建、挂机刷怪、日常维护等全流程操作，无需人工干预。

### ✨ 核心功能

| 功能 | 说明 |
|------|------|
| **自动代练** | 下单后自动注册账号、创建角色（金灵根100）、装备技能/功法/武器、开启自动刷怪 |
| **智能升级** | 每 15 分钟检查经验，自动升级直到 120 级满级或经验不足 |
| **健康监测** | 每 30 分钟检查账号状态，自动补装技能/功法/装备，掉线自动重连刷怪 |
| **仙盟日常** | 每日自动加入仙盟、灵池沐浴、仙园采摘、悟道树冥想、洞府采集 |
| **每日试炼** | 每日自动完成试炼副本，获取试炼币收益 |
| **全自动交付** | 账号全部满级后自动完成工单，通知用户 |

## ⚙️ 自动工作流体系

平台后端由 **7 个 GitHub Actions 工作流** 驱动，全自动化运行：

| 工作流 | 频率 | 功能 |
|--------|------|------|
| 📋 **订单扫描** | 每 10 分钟 | 扫描已审批工单，自动注册账号并创建角色 |
| ⬆️ **一键升级** | 每 15 分钟 | 检查所有挂机账号经验，循环升级直到满级 |
| ❤️ **健康监测** | 每 30 分钟 | 检查技能/功法/装备/刷怪状态，自动修复 |
| 🏯 **每日维护** | 每日 0:00 | 仙盟日常 + 洞府采集 |
| 🗡️ **每日试炼** | 每日 2:00 | 自动完成试炼副本 |
| 🔬 **试炼测试** | 每 3 天 | 穷举试炼词条组合，优化收益 |
| 🚀 **自动部署** | 代码推送 | 自动部署到 Cloudflare |

## 🛡️ 安全特性

- **反检测技术** — 每账号独立 IP 轮换、随机机器码、浏览器指纹伪装
- **操作延迟模拟** — 智能随机延迟、分段暂停，行为模式接近真实玩家
- **异常监控** — 自动检测账号异常状态并记录，用户可实时查看
- **数据加密** — 游戏账号密码安全存储，通过加密 API 传输
